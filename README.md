# :rocket: Getting started:

- 1 ) Cloning the repo:
```
git clone https://github.com/Keyj33k/Ovax.git
```

- 2 ) Navigate to the Ovax directory.
```
cd Ovax
```

- 3 ) You need to install the requirements:
```
pip install -r requirements.txt
```

- 4 ) Now you can run IPSweep:
```
python3 ovax.py 
```
